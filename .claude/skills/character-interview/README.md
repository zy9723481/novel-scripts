## Quick Start

This skill activates automatically when you:

- Create new characters for a screenplay
- Deepen understanding of existing characters
- Build character backstory and motivation
- Develop character voice and mannerisms

## What This Skill Does

Provides the 80-question character interview framework for deep character
development covering background, psychology, relationships, habits, and
motivations to build comprehensive character backstories.
